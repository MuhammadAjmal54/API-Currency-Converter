# 🌍 API Currency Converter 💱

A simple **currency converter** built using **HTML, CSS, and JavaScript**, utilizing an external API to fetch real-time exchange rates.

## 🚀 Features
- **Live Exchange Rates** – Fetches up-to-date currency conversion rates.
- **User-Friendly Interface** – Simple and clean design.
- **Supports Multiple Currencies** – Convert between various global currencies.
- **Fast & Lightweight** – Optimized for quick performance.

## 📂 Project Structure
```
📦 currency-converter
├── 📄 index.html    # Main HTML file
├── 🎨 style.css     # Styling for the UI
├── ⚡ app.js        # Fetches exchange rates & handles conversion logic
└── 🌍 codes.js      # Stores currency codes & related data
```

## 📌 How to Use
1. Open `index.html` in a browser.
2. Select currencies and enter the amount.
3. Click "Convert" to see real-time exchange results.

## 🛠️ Setup Instructions
1. **Clone the repository**  
   ```bash
   git clone https://github.com/your-username/currency-converter.git
   cd currency-converter
   ```
2. **Open the project in a browser**  
   Just open `index.html` – no additional setup required!
